# Infinity Saturation

A Minecraft Bedrock Edition behavior pack that grants infinite saturation to all players.

## Features

- Continuously applies saturation effect (level 255, infinite duration) to all players
- Hunger bar no longer depletes from sprinting or other activities
- Particles hidden for a cleaner experience

## Requirements

- Minecraft Bedrock Edition 1.21.0 or later

## Installation

1. Download the `.mcpack` file
2. Open it to import into Minecraft
3. Enable the pack **World Settings -> Behavior Packs**
4. Activate the world with the pack
