# Night Vision

A Minecraft Bedrock Edition behavior pack that grants infinite night vision to all players.

## Features

- Continuously applies night vision effect (level 255, infinite duration) to all players
- See clearly in dark environments

## Requirements

- Minecraft Bedrock Edition 1.21.0 or later

## Installation

1. Download the `.mcpack` file
2. Open it to import into Minecraft
3. Enable the pack **World Settings -> Behavior Packs**
4. Activate the world with the pack
