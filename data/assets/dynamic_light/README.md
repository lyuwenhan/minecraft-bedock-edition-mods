# Dynamic Light

A Minecraft Bedrock Edition behavior pack that adds dynamic lighting effects for light-emitting items held in the player's hand.

## Features

When a player holds a light-emitting item in their main hand, it creates ambient light around them. The light updates as the player moves and disappears when they leave the area.

## Light Level Reference

| Item | Light Level |
|:-:|:-:|
| Beacon, Campfire, Glowstone, Jack o'Lantern, Lava Bucket, Sea Lantern, Conduit, Lantern, Shroomlight, Froglight, End Rod | 15 |
| Glow Berries, Torch | 14 |
| Soul Torch, Soul Lantern, Crying Obsidian, Soul Campfire | 10 |
| Enchanting Table, Ender Chest, Redstone Torch, Glow Lichen | 7 |
| Sea Pickle, Sculk Catalyst | 6 |
| Magma Block | 3 |
| Firefly Bush | 2 |
| Brewing Stand, Brown Mushroom, Dragon Egg, Sculk Sensor | 1 |
| Light Block (light_block_0 ~ light_block_15) | 0–15 |

## Requirements

- Minecraft Bedrock Edition 1.21.0 or higher

## Installation

1. Download the `.mcpack` file
2. Open it to import into Minecraft
3. Enable the pack **World Settings -> Behavior Packs**
4. Activate the world with the pack
